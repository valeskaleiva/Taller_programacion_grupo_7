# 🃏 Sistema de Gestión TCG 

## 📌 Descripción del Proyecto

Proyecto final de la carrera **Analista Programador de Duoc UC**.
Consiste en el desarrollo de una aplicación web para la gestión y automatización de procesos de ventas de la tienda especializada en Trading Card Games (TCG).

La solución busca optimizar el control de stock, el registro de ventas y la generación de reportes, reemplazando procesos manuales por un sistema digital eficiente.

---

##  Objetivo General

Desarrollar una aplicación web que permita mejorar la gestión del negocio, facilitando el control de inventario, ventas y acceso a información relevante para la toma de decisiones.

---

##  Funcionalidades Principales

* Gestión de productos y stock
* Registro de ventas
* Administración de usuarios
* Generación de reportes

---

##  Arquitectura

El sistema se basa en una arquitectura **cliente-servidor**, donde:

* El cliente accede mediante un navegador web
* El servidor gestiona la lógica de negocio y base de datos

---

##  Tecnologías Utilizadas

* .NET
* Base de datos SQL
* HTML, CSS, JavaScript
* Git y GitHub
